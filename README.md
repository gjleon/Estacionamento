# 🚗 Sistema de Estacionamento

Um sistema simples de estacionamento desenvolvido em C# para gerenciar entrada e saída de veículos via console.

---

## 📌 Sobre o Projeto

Este projeto permite:

* Adicionar veículos ao estacionamento
* Remover veículos e calcular o valor a ser pago
* Listar todos os veículos estacionados

O cálculo do valor é baseado em um preço inicial + valor por hora.

---

## 🚀 Funcionalidades

✅ Cadastro de veículos (placa)
✅ Remoção de veículos
✅ Cálculo automático de valor
✅ Listagem de veículos estacionados

---

* C#
* .NET
* Console Application
